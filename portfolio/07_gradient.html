Project Gradient Noise (Perlin)

3D Graphics · WebGL

Project Overview

Gradient noise (Perlin noise) improves on value noise by assigning pseudo-random gradient vectors to lattice points and computing the dot product between the gradient and the offset vector from the lattice point. This eliminates the grid artifacts of value noise, producing organic patterns used extensively in procedural terrain, clouds, and textures.

Tasks Completed

  • Implemented a 2D Perlin noise function in GLSL using a permutation table encoded as a texture.
  • Computed gradient dot products and interpolated with the quintic fade curve.
  • Extended to 3D Perlin noise for volumetric effects.
  • Demonstrated fBm layering and domain-warping using the noise as an offset into itself.
  • Compared side-by-side output with value noise to highlight the reduction in grid artifacts.

Reflection

Domain warping — feeding noise output back into the noise input coordinates — creates spectacularly organic shapes with almost no extra code. It was a breakthrough moment in understanding how simple operations compose into rich visual complexity. This technique directly applies to procedural terrain and fire effects.
