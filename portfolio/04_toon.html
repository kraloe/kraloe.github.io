Project Toon / Cell Shading

3D Graphics · WebGL

Project Overview

Toon (cell) shading replaces smooth Phong lighting with discrete colour bands to produce a hand-drawn cartoon aesthetic. This project implements the technique in a GLSL fragment shader by quantising the diffuse intensity into N discrete steps and optionally adding a silhouette outline pass using back-face inflation in the vertex shader.

Tasks Completed

  • Computed diffuse intensity using the Lambertian dot-product (N · L) and quantised it into configurable discrete bands.
  • Added a specular highlight clamp to produce a single hard specular dot.
  • Implemented an outline pass by rendering back-faces scaled outward along normals with a solid dark colour.
  • Exposed sliders for band count, outline thickness, and light direction.
  • Compared output against standard Phong shading to validate the stylistic difference.

Reflection

The trickiest part was the silhouette pass — scaling geometry along normals in clip space rather than world space avoids distortion on non-uniform objects. Getting the band quantisation to look clean (not aliased) required a small smoothstep around each threshold, which blends between hard bands.
