Project Fog

3D Graphics · WebGL

Project Overview

This project implements two classic fog models in a GLSL fragment shader: Linear fog (remapping depth between near and far planes) and Exponential fog (using an e^(-depth*density) falloff). Both modes blend the fragment colour towards a configurable fog colour based on camera-space depth, simulating atmospheric scattering effects seen in many real-time games.

Tasks Completed

  • Implemented Linear fog by computing a blend factor from the fragment's eye-space Z depth.
  • Implemented Exponential fog with a density parameter, including the squared-density variant (exp2).
  • Added an interactive UI to switch between fog modes and adjust density / near-far range in real time.
  • Ensured fog is computed in eye/view space to remain camera-independent.
  • Tested on multiple scene objects to validate correct depth ordering.

Reflection

The subtlety here is choosing the right space to compute depth — computing it in world space rather than eye space breaks when the camera moves. Understanding the difference between linear and exponential falloff and when each looks more realistic was a valuable lesson in physically plausible shading.
